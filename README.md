# Adolescent HIV Resource Hub

## Mission Statement

This collaborative, open-source resource guide aims to provide adolescents at risk for or living with HIV, as well as their healthcare providers, with reliable, up-to-date information on local HIV testing sites, youth-friendly clinics, mental health support, PrEP access points, and educational materials.

Our goal is to reduce barriers to care, empower adolescents with knowledge, and foster a community-driven approach to improving health outcomes. By leveraging transparent collaboration on GitHub, we ensure that the guide remains current, accurate, and responsive to the needs of the communities we serve.

## How to Use This Guide

The main resource list is available in [resources.md](resources.md). It is organized by category (testing centers, clinics, mental health services, PrEP providers, educational materials) and presented in table format for easy scanning.

## How to Contribute

We welcome contributions from healthcare providers, community members, and anyone with verified information about relevant resources.

### Suggesting a New Resource

1. Go to the [Issues](https://github.com/adisofand/adolescent-hiv-resource-hub/issues) tab.
2. Click **New Issue** and select the **"Suggest a New Resource"** template.
3. Fill out the requested details (resource name, type, location, contact, description, website).
4. Submit the issue. It will be labeled `new-suggestion` and reviewed by the maintainers.

### Reporting an Outdated Entry

1. Open an issue using the **"Report an Outdated Entry"** template.
2. Provide the name of the resource and describe what information is incorrect or outdated.
3. Submit the issue. It will be labeled for verification and correction.

### Directly Editing the Resource List

If you are comfortable with Git and GitHub workflows:

1. Fork this repository.
2. Create a new branch for your changes.
3. Edit `resources.md` with the new or updated information.
4. Open a Pull Request (PR) referencing the related issue (if any).
5. A maintainer will review and merge your changes after verification.

## Review Process

All contributions go through a verification process to ensure accuracy and reliability. Resources are checked for current contact details, eligibility criteria, and appropriateness for adolescent populations.
